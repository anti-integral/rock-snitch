# data/relevant/ - curated Mars2020 Navcam frames

Hand-picked subset from the full ~1500-sol dataset for testing, demos, and
lightweight checkout-only samples. The full `data/images/` and `data/metadata/`
trees are gitignored (100+ GB), so this folder is what tests + the Gradio
example gallery actually use.

## Re-creating this set

```bash
# 1. Download the full dataset (writes data/images/, data/metadata/)
rock-snitch fetch data/

# 2. Build the stereo index
rock-snitch index data/

# 3. Pick frames you want and copy under data/relevant/
```
